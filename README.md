Este es un proyecto de aplicación web que utiliza React, con un backend Nodejs que utiliza Express, JWT,  Material-UI   y Mongodb como base de datos

### Implementación

sh
git clone
cd Gestion-Eventos
cd client 
npm i
 
### Frontend
sh
cd Gestion-Eventos
cd client
npm run dev
 
### Backend 
sh
cd Gestion-Eventos
npm run dev



>Necesita tener una base de datos Mongodb ejecutándose
